Sistem Kasir Warung

Aplikasi manajemen transaksi sederhana untuk usaha mikro berbasis Python.

Python Management Accounting

Tentang Proyek

Sistem Kasir Warung adalah program aplikasi Python sederhana yang dirancang untuk membantu pemilik warung atau usaha kecil dalam mencatat pesanan, menghitung total belanja, dan memproses pembayaran secara otomatis melalui antarmuka baris perintah (CLI).

Fitur Utama

  • Daftar Menu Dinamis: Menyimpan data menu dan harga dalam struktur data dictionary.
  • Sistem Keranjang Belanja: Konsumen dapat memesan banyak jenis item sekaligus.
  • Perhitungan Otomatis: Menghitung total harga per item dan total akhir belanjaan secara akurat.
  • Struk & Kembalian: Menampilkan ringkasan pembayaran dan menghitung uang kembalian secara otomatis.

Penjelasan Kode (Logic)

Aplikasi ini menggunakan perulangan `while` untuk input pesanan terus-menerus hingga pengguna selesai, serta `try-except` untuk menangani input angka yang tidak valid. Berikut kode lengkapnya: 

menu = {
    "1" : ["ayam bakar", 15000],
    "2" : ["lele bakar", 10000],
    "3" : ["ayam goreng", 12000],
    "4" : ["nasi goreng", 10000]
}

keranjang = []

print("="*30)
print("SELAMAT DATANG DI WARUNG SAYA")
print("SILAKAN PILIH MENUNYA")
print("="*30)

print("DAFTAR MENU")
for kode, item in menu.items() :
    print (f"{kode}. {item[0]} - Rp{item[1]}")

while menu :
    kode = input("Masukkan kode menu yang anda pesan atau (q) untuk keluar : ")

    if kode.lower() == "q" :
        break 
    if kode not in menu :
        print("Masukkan kode yang ada di menu")
        continue
    try:
        jumlah = int(input("Masukkan jumlah : "))
    except ValueError:
        print("Masukkan jumlah yang benar (angka)")
        continue
        
    nama = menu[kode][0]
    harga = menu[kode][1]
    total_harga = harga * jumlah  
    keranjang.append([nama, harga, jumlah, total_harga])
    print(f"{jumlah} x {nama} berhasil ditambahkan.")

print("="*30)
print("STRUK PEMBAYARAN")
total_bayar = 0 
for item in keranjang :
    print(f"{item[0]} x {item[2]} - Rp{item[3]}")
    total_bayar += item[3]

print(f"Total Bayar : Rp{total_bayar}")

while True :
    try:
        uang = int(input("Uang dibayar : Rp"))
        if uang < total_bayar:
            print("Uang kurang! Coba lagi")
        else:
            break
    except ValueError:
        print("Masukkan angka yang benar")

kembalian = uang - total_bayar
print(f"Kembalian : Rp{kembalian}")
print("Terima kasih sudah berbelanja")

Tantangan & Pelajaran

Proyek ini mengajarkan saya dasar logika bisnis pemrograman: Create (membuat pesanan), Read (baca menu), dan perhitungan aritmatika sederhana. Mengelola input user agar program tidak crash saat dimasukkan huruf (bukan angka) adalah tantangan tersendiri yang saya atasi dengan blok `try-except`.
