Rocket Escape

Game arkade bertahan hidup berbasis Python & Pygame.

Python Pygame Library Desktop App

Tentang Proyek

Rocket Escape adalah game sederhana namun menantang yang saya buat untuk melatih logika pemrograman berorientasi objek (OOP) di Python. Tujuan game ini adalah bertahan hidup selama mungkin dengan menghindari asteroid yang jatuh dengan kecepatan yang terus meningkat.

Fitur Utama

  • Kontrol Responsif: Pemain dapat menggerakkan roket ke kiri dan kanan dengan mulus.
  • Kesulitan Dinamis: Semakin lama bertahan, asteroid akan jatuh semakin cepat.
  • Sistem Skor & Highscore: Menyimpan skor tertinggi secara lokal.

Penjelasan Kode (Logic)

Inti dari game ini adalah "Game Loop" yang berjalan terus menerus untuk memperbarui posisi objek di layar. Berikut adalah cuplikan logika bagaimana asteroid dimunculkan secara acak: 

# Logika spawning asteroid
def spawn_asteroid(asteroids, screen_width):
    if len(asteroids) < MAX_ASTEROIDS:
        x_pos = random.randint(0, screen_width - 50)
        y_pos = -50 # Muncul di atas layar
        asteroids.append(pygame.Rect(x_pos, y_pos, 50, 50))
    
    # Logika pergerakan dan tabrakan
    for asteroid in asteroids:
        asteroid.y += FALL_SPEED
        
        # Deteksi tabrakan dengan roket
        if player_rect.colliderect(asteroid):
            game_over = True

Saya menggunakan modul `random` untuk menentukan posisi X asteroid, dan `pygame.Rect` untuk menangani deteksi tabrakan (`colliderect`) yang akurat antara kotak roket dan asteroid.

Tantangan & Pelajaran

Tantangan terbesar dalam proyek ini adalah menyeimbangkan tingkat kesulitan (game balancing) agar tidak terlalu mudah di awal tetapi tetap bisa dimainkan di level tinggi. Saya belajar banyak tentang manajemen memori (menghapus objek asteroid yang sudah keluar layar) dan optimalisasi loop rendering.
