Password Generator

Aplikasi CLI Python untuk menghasilkan dan memeriksa kekuatan password.

Python Security CLI App

Tentang Proyek

Password Generator & Checker adalah alat berbasis command web-line interface (CLI) yang dirancang untuk membantu pengguna membuat kata sandi yang kuat dan aman. Selain itu, alat ini juga memiliki fitur untuk menilai kekuatan kata sandi yang sudah ada.

Fitur Utama

  • Generate Password: Membuat password acak dengan kombinasi huruf, angka, dan simbol.
  • Cek Kekuatan Password: Menganalisis password berdasarkan panjang dan kompleksitas karakter.
  • Validasi Input: Memastikan panjang password minimal 8 karakter.

Penjelasan Kode (Logic)

Program ini menggunakan library built-in `random` dan `string` untuk menghasilkan karakter acak. Berikut adalah kode lengkapnya: 

import random
import string

def generate_password(length=12):
    if length < 8:
        return "Panjang password minimal 8 karakter."

    characters = string.ascii_letters + string.digits + string.punctuation
    password = ''.join(random.choice(characters) for _ in range(length))
    return password

def check_password_strength(password):
    strength = 0

    if len(password) >= 8:
        strength += 1
    if any(char.islower() for char in password):
        strength += 1
    if any(char.isupper() for char in password):
        strength += 1
    if any(char.isdigit() for char in password):
        strength += 1
    if any(char in string.punctuation for char in password):
        strength += 1

    if strength <= 2:
        return "Password lemah ❌"
    elif strength == 3 or strength == 4:
        return "Password cukup kuat ⚠️"
    else:
        return "Password sangat kuat ✅"

def main():
    print("=== PASSWORD GENERATOR & CHECKER ===")
    print("1. Buat Password")
    print("2. Cek Kekuatan Password")
    
    pilihan = input("Pilih menu (1/2): ")

    if pilihan == "1":
        panjang = int(input("Masukkan panjang password: "))
        password = generate_password(panjang)
        print("Password yang dihasilkan:", password)

    elif pilihan == "2":
        password = input("Masukkan password yang ingin dicek: ")
        hasil = check_password_strength(password)
        print("Hasil:", hasil)

    else:
        print("Pilihan tidak valid.")


if __name__ == "__main__":
    main()

Fungsi `check_password_strength` bekerja dengan memberikan skor (0-5) berdasarkan kriteria keamanan seperti adanya huruf besar, huruf kecil, angka, dan simbol.
